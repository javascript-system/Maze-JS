Acessibilidade

Caso ativado, as fases não têm limite de blocos no Blockly.
Caso ativado, O usuário usar qualquer tipo de bloco mesmo fora de sua fase. Aviso: caso isso seja ativado, a fase atual não será alterada para o usuário não perder progresso.
Ao iniciar uma fase, se ela tiver um texto de introdução, ele não será mostrado se isso for desativado.
