Aprenda programação jogando!

Desvende desafios, crie algoritmos e domine a lógica de programação de forma prática, interativa e totalmente visual.

💡 Como funciona

No jogo, você irá usar comandos de programação reais ou blocos de código para guiar seu personagem através de labirintos.

Para garantir a segurança do site e dos usuários, utilizamos tecnologias de Worker, que isolam os códigos do ambiente principal. Os métodos de programação são simples e acessíveis: iniciantes podem começar usando o Google Blockly, encaixando blocos para aprender os conceitos básicos, e depois evoluir para o JavaScript, onde poderão escrever loops, funções e lógicas mais avançadas.

🎯 Objetivos do Jogo

O jogo foi desenvolvido como uma plataforma de aprendizado voltada para iniciantes na programação. Nosso objetivo é claro: não queremos que os usuários fiquem presos jogando indefinidamente, mas sim que reforcem suas habilidades e conhecimentos. Por meio do jogo, os jogadores irão aprender conceitos básicos de programação de forma interativa e divertida, preparando-se para avançar para linguagens mais sofisticadas e projetos reais no futuro.
